Проект "СЕРВЕР"

Этот проект создан как решение тестового задания по адресу: https://github.com/drumanew/plgn_db

Для сборки проекта достаточно запустить команду make. При этом будут установлены следующие компоненты:
1) сервер БД REDIS 5.0.5
2) библиотека HIREDIS для упрощенного взаимодействия с сервером REDIS
3) библиотека zeromq 4.3.1

А также произойдёт компилляция файлов server.c, client.c и subscriber.c соотвественно в srv, clt, sub в каталог /src
    - файл server.c содержит исходный код сервера, взаимодействующего с БД REDIS, клиентами и подписчиками
    - файл client.c содержит упрощенный вариант клиента для проверки сервера
    - файл subscriber.c содержит упрощенный вариант подписчика уведомлений для проверки сервера

Вся работа с сервером и взаимодействие с ним осуществляется согласно ТЗ, описанному на сайте
